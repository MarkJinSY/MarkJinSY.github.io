Java基本数据类型

基本类型 大小(字节) 默认值 封装类
byte 1 (byte)0 Byte
short 2 (short)0 Short
int 4 0 Integer
long 8 0L Long
float 4 0.0f Float
double 8 0.0d Double
boolean - false Boolean
char 2 \u0000(null) Character
  • int为基础数据类型，Integer为引用类型，因而Integer的默认值为null，在使用前需先为其指定一个对象，否则会报错；
  • 基本数据类型在声名时系统会自动为其分配空间，而引用类型声明时只是分配了引用空间，必须实例化开辟数据空间后才可以为其赋值。将一个数组赋值给另一个数组时只是复制了一个引用，对一个数组的操作，在另一个数组中也可见。

装箱和拆箱

装箱即基本数据类型转换成包装器类型(int -> Integer)；调用方法：Integer = valueOf(int)

拆箱即包装器类型转基本数据类型，(Integer -> int); 调用方法：int = intValue

Java SE5之前，生成一个值为10的Integer对象：
Integer i = new Integer(10);
Java SE5之后提供了自动装箱的特性，生成一个数值为10的Integer对象：
Integer i = 10;

equals和==

  • equals：
    • 值比较，用于比较两个对象的内容是否相等，由于所有类都继承自Object因而适用于所有对象。如果没有覆盖该方法的话，调用的仍是Object类中的方法，而Object类中的equals方法返回的是==的判断。
    • 使用equals进行比较时，把常量写在前，因为Object的equals object可能为null。
  • ==：
    • 比较的是变量()内存中存放的对象的()堆内存地址，用于判断两个对象的地址是否相同，即是否指向同一个对象。
    • 两边的操作数必须是同一类型或父子类之间才能编译通过。
    • 当比较值为数字时，值相等即为true，如int a=10, Long b=10L, double c=10.0都是相等的，因为他们都指向地址为10的堆。

+=操作符

+= 操作符进行隐式自动类型转换，a+=b将隐式的加操作结果类型强制转换成为持有的结果类型，a=a+b则不会自动进行类型转换。
